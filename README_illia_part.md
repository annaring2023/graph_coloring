## Процес виконання і використання

**Встановлення залежностей:**

Перед запуском програми переконайтеся, що встановлено необхідні бібліотеки Python. Для цього виконайте команду:

Bash
```
pip install networkx 
pip install matplotlib
```

**Запуск програми:**

Взаємодія з програмою відбувається через командний рядок (CLI). Основний синтаксис виглядає так:

Bash
```
python main.py <шлях_до_файлу> [опції]
```
